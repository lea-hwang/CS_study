## 업로드 규칙
- 강의 시청 후, 정리된 내용은 '{차시번호}_{차시 제목}.md' ex) `1_운영체제 개요.md`
- 모든 이미지 파일은 해당 과목 내 `assets` 폴더 내부에 넣어주세요.
- commit message는 '{발표 날짜} / {이름} / {강의 제목}' ex) '2023.03.14 / 황희원 / 2_컴퓨터시스템'
